The execline library interface

The execline library has been designed to help programmers write applications suitable for use in execline scripts.

Compiling

  • Add /package/admin/execline/include, or wherever you installed your header files, to your header directory list
  • Use #include "execline.h"

Linking

  • For static linking: add /package/admin/execline/library and /package/prog/skalibs/library, or wherever you installed your execline and skalibs .a files, to your library directory list.
  • For dynamic linking: add /package/admin/execline/library.so and /package/prog/skalibs/library.so, or wherever you installed your execline and skalibs .so files, to your library directory list - and make sure to properly edit your ld.so.conf file and to run ldconfig if needed.
  • Link with -lexecline and -lstddjb

Programming

Check the execline.h header for the exact function prototypes.

The API will be documented when it is really stabilized.
